About Page Project Clean Tweets: This program performs sentiment analysis with the intention of determining how positive or negative a particular user is over a period of time. The analysis is completed by analyzing the language used the tweets for positive or negative value. Expert Mode: Users enter a twitter handle to retrieve the most recent tweet from the handle's profile. The sentiment score can be viewed for the past 25, 50, 100, or 200 tweets from an account. Battle Mode: Allows users to enter two twitter handles. Comparison is made between sentiment of each handle. User can select a graphical display of Overall sentiment score, Positive sentiment score, or Negative sentiment score.
